# Election_Analysis

## Project Overview

The following analysis is an election audit of a recent local congressional election for the Colorado Board of Elections. The code needed to analyze the data needed to do the follow:

  1. Calculate the total number of votes cast.
  2. Get a complete list of candidates who received votes.
  3. Calculate the total number of votes each candidate received.
  4. Calculate the percentage of votes each candidate won.
  5. Determine the winner of the election based on the popular votes. 

## Resources

Data source: election_results.csv<br/>
Softward: Python 3.7.6, Visual Studio Code 1.68.1

## Summary

#### Candidate Results
Charles Casper Stockham received 23.0% of the vote with 85,213 number of votes.<br/>
Diana DeGette received 73.8% of the vote with 272,892 number of votes.<br/>
Raymon Anthony Doane received 3.1% of the vote with 11,606 number of votes.<br/>

#### Winner
Diana DeGette won the election with 73.8% of the vote and 272,892 number of votes.
